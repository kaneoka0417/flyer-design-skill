# チラシ vs Webランディングページ

## ⚠️ 最大の注意点

**チラシはWebサイトではない。** HTMLで作るからといってLP風にしてはいけない。

## 根本的な違い

| 項目 | 印刷チラシ | Web LP |
|------|-----------|--------|
| **密度** | 高い（詰め込む） | 低い |
| **余白** | 少ない（もったいない） | 多い（意図的に活用） |
| **レイアウト** | 分割・ブロック型 | 縦スクロール型 |
| **閲覧パターン** | 面的（興味ある所に目が飛ぶ） | 線的（上から下へ誘導） |
| **インパクト** | 装飾的（派手に目立つ） | 内容的（共感・ストーリー） |
| **制約** | A4等の固定サイズに収める | 縦に無限に伸ばせる |

## チラシがLPっぽくなる失敗パターン

❌ 余白を取りすぎる
❌ セクションごとにスペースを空けすぎる
❌ 情報を「整理」しすぎる
❌ ストーリーテリング型の縦並び構成
❌ シンプルすぎて目を引かない

## 日本の商業チラシの特徴的要素

### 装飾
- **破裂マーク（爆発）**: 価格、「限定」、「新発売」に使用
- **集中線**: 注目させたい箇所に
- **キラキラ**: 高級感、お得感
- **吹き出し**: コメント、補足

### 文字
- **極太ゴシック**: インパクト重視
- **手書き風フォント**: 親しみやすさ
- **POP文字**: 温かみ、目を引く
- **擬音語**: 「ドン！」「ガツン！」「もちもち」

### レイアウト
- **情報を詰め込む**: スペースを無駄にしない
- **縦横混在テキスト**: レイアウト優先
- **ブロック分割**: 情報をグループ化
- **斜め配置**: 動きを出す

### 必須要素
- **地図**: 実店舗への誘導
- **QRコード**: Webへの誘導
- **電話番号**: 大きく目立つように

## A4に収める情報の優先順位

### 優先度【高】（必須）
1. **WHAT**: イベント名、商品名（最大・最初に目に入る）
2. **BENEFIT**: 得られる価値（50%OFF、プレゼント等）
3. **WHEN/WHERE**: 日時、場所、地図
4. **HOW**: 連絡先、QRコード

### 優先度【中】（あると良い）
- WHO: 主催者情報
- TARGET: ターゲットを明確にするコピー
- 詳細情報: スペック、タイムテーブル

### 優先度【低】（スペースがあれば）
- 補足情報: 会社概要など
- イメージ写真: 雰囲気を伝えるもの

## ⚠️ A4に収まらない時の対処法

**はみ出る = 情報が多すぎる**

削る優先順位（下から削る）：
1. 会社概要、沿革 → 削除
2. 詳細な説明文 → 箇条書き1行に
3. 「こんな方に」セクション → 削除 or 1行に
4. プログラム詳細 → 簡略化
5. 複数の特典 → 最重要1-2個に

**残すべきもの（絶対削らない）：**
- タイトル/キャッチコピー
- 日時・場所
- 価格/参加費
- 連絡先/QRコード

## 面積配分の目安（A4）

```
┌─────────────────────────────┐
│  キャッチ・タイトル  (20-25%)│
│  最大インパクト、ここで掴む  │
├─────────────────────────────┤
│                             │
│  メインコンテンツ  (50-55%) │
│  特徴、ベネフィット         │
│  ★情報は厳選！詰めすぎない  │
│                             │
├─────────────────────────────┤
│  日時・場所・連絡先 (20-25%)│
│  QRコード + 電話 + 価格     │
└─────────────────────────────┘
```

**A4縦の実測：297mm = 約842px(@72dpi)**
padding 8mm×2 = 16mm引くと、使える高さは約281mm

## CSS実装のポイント（超重要）

```css
/* ===== 絶対守る：A4固定 ===== */
body {
  width: 210mm;
  height: 297mm;
  overflow: hidden; /* はみ出し禁止 */
  margin: 0;
  padding: 0;
}

/* ===== 縦方向を目一杯使う ===== */
.container {
  height: 100%;
  display: flex;
  flex-direction: column;
}

/* メインコンテンツを伸ばして余白を埋める */
.main-content {
  flex: 1;  /* 残りスペースを全部使う */
}

/* または各セクションを均等配置 */
.container {
  justify-content: space-between;
}

/* ===== 余白は最小限 ===== */
.container {
  padding: 5mm;        /* Web: 15-20mm → チラシ: 5-6mm */
}

/* ===== 要素間は詰める ===== */
.section {
  margin-bottom: 3mm;  /* Web: 15-20mm → チラシ: 2-4mm */
}

.grid {
  gap: 2mm;            /* Web: 10-15mm → チラシ: 2-3mm */
}

.block {
  padding: 3mm;        /* Web: 10-15mm → チラシ: 3-4mm */
}

/* ===== 行間は詰める ===== */
body {
  line-height: 1.3;    /* Web: 1.6-1.8 → チラシ: 1.2-1.4 */
}

h1, h2, h3 {
  line-height: 1.1;    /* 見出しはもっと詰める */
  margin: 0 0 1mm 0;
}

/* ===== フォントサイズ目安 ===== */
.main-title { font-size: 24-28pt; }  /* 最大 */
.section-title { font-size: 11-13pt; }
.body-text { font-size: 8-9pt; }     /* 小さめでOK */
.caption { font-size: 7pt; }         /* 注釈 */

/* ===== リストは詰める ===== */
ul, ol {
  margin: 0;
  padding-left: 4mm;
}
li {
  margin-bottom: 0.5mm; /* ほぼ詰める */
}
```

### Web vs チラシ 数値比較

| 項目 | Web LP | 印刷チラシ |
|------|--------|-----------|
| container padding | 15-20mm | **5-6mm** |
| section margin | 15-20mm | **2-4mm** |
| grid gap | 10-15mm | **2-3mm** |
| block padding | 10-15mm | **3-4mm** |
| line-height | 1.6-1.8 | **1.2-1.4** |
| 本文フォント | 14-16px | **8-9pt** |
