# アイコン・イラスト・画像ガイド

## フリーアイコン（商用可）

### ⚠️ 絵文字は非推奨
絵文字（📚🛠️✨）は**印刷チラシには不向き**：
- 環境依存で見た目が変わる
- 印刷時に色がずれる/潰れる
- プロっぽくない

**→ SVGアイコンかイラストを使う**

### SVGアイコン（推奨）
高品質、サイズ自由、色も変更可能。印刷に最適。

**Heroicons** (MIT)
- https://heroicons.com/
- シンプル、モダン、ビジネス向け

**Tabler Icons** (MIT)
- https://tabler-icons.io/
- 4000+アイコン、線が細め

**Lucide** (ISC)
- https://lucide.dev/
- Feather Iconsの後継

**Font Awesome** (部分無料)
- https://fontawesome.com/
- 定番、種類豊富

### 使い方（インラインSVG）
```html
<svg width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
  <path d="M12 2L2 7l10 5 10-5-10-5z"/>
  <path d="M2 17l10 5 10-5"/>
  <path d="M2 12l10 5 10-5"/>
</svg>
```

### 使い方（imgタグ）
```html
<img src="icon.svg" width="24" height="24" alt="">
```

## チラシでの使い方

### ビジュアル要素のバランス

**良いチラシ = アイコン + イラスト + 写真 のミックス**

```
┌─────────────────────────────────┐
│ [写真] キャッチ・ヘッダー部分   │  ← 写真で目を引く
├─────────────────────────────────┤
│ 🔧 特徴1    🎯 特徴2            │  ← アイコンで区別
│ [イラスト] [イラスト]           │  ← イラストで親しみ
├─────────────────────────────────┤
│ [人物写真] お客様の声           │  ← 写真で信頼感
└─────────────────────────────────┘
```

### 使い分け

| 種類 | 役割 | 配置場所 |
|------|------|----------|
| **写真** | 信頼感、リアリティ | ヘッダー、人物紹介、実績 |
| **イラスト** | 親しみ、わかりやすさ | 説明、フロー図、特徴 |
| **アイコン** | 区別、視線誘導 | 見出し横、リスト、ラベル |

### アイコンの配置
```
✅ 見出しの横に小さく
   [icon] 学べること

✅ リストのブレット代わり
   ✓ ソースコード提供

✅ 番号の代わりに丸囲み
   ① → ② → ③

❌ アイコンだらけ（うるさい）
❌ 写真の代わりにアイコンだけ
```

### イラストの配置
```
✅ フロー図の各ステップ
   [イラスト] → [イラスト] → [イラスト]

✅ 特徴ブロックの中央
   ┌─────────┐
   │[イラスト]│
   │  特徴1  │
   └─────────┘

✅ 吹き出しの中
```

### 写真の配置
```
✅ ヘッダー背景（暗くして文字載せ）
✅ 人物（講師、お客様の声）
✅ 会場・商品のイメージ
✅ ビフォーアフター

❌ 低解像度（72dpi）はNG
❌ 素人っぽい写真
```

### サイズの目安
- アイコン（見出し横）: 16-24pt
- アイコン（ブレット）: 10-12pt  
- イラスト（特徴ブロック）: 15-25mm
- 写真（メイン）: 幅50%以上

## 画像の注意点

### 解像度
- 印刷用: **350dpi以上**
- Web用72dpiはNG（ぼやける）

### ファイル形式
- 写真: JPG（軽い）
- イラスト/アイコン: SVG or PNG（透過）

### 著作権
- 必ずライセンス確認
- クレジット必要な場合あり
- 加工可否を確認
