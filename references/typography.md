# 日本語タイポグラフィガイド

## フォント選びの基本

### ルール
- 使用フォントは**3種類以内**
- 多用すると散らかって読みにくい

### フォントの種類と印象

| 種類 | 印象 | 用途 |
|------|------|------|
| 明朝体 | 伝統的、高級、真面目、女性的 | 長文、高級志向 |
| ゴシック体 | モダン、親しみやすい、力強い | 見出し、強調 |
| 丸ゴシック | 柔らかい、優しい、可愛い | 子供向け、カジュアル |
| 毛筆体 | 和風、インパクト、手作り感 | アクセント、和風デザイン |

### ターゲット別おすすめ

| ターゲット | おすすめ |
|------------|----------|
| 高級志向 | 明朝体 |
| ファミリー層 | ゴシック体、丸ゴシック |
| 若者 | ゴシック体、デザイン書体 |
| 高齢者 | ゴシック体（太め） |
| 子供 | 丸ゴシック |
| ビジネス | ゴシック体 |

## 推奨フォント（フリー/商用可）

### ゴシック体
- **Noto Sans JP** - Google提供、汎用性高い
- **源ノ角ゴシック** - Adobe/Google提供
- **M PLUS 1p** - 丸みのあるゴシック

### 明朝体
- **Noto Serif JP** - Google提供
- **源ノ明朝** - Adobe/Google提供

### 丸ゴシック
- **M PLUS Rounded 1c**
- **コーポレート・ロゴ丸**

### CSS指定例
```css
body {
  font-family: 
    "Noto Sans JP",
    "Hiragino Kaku Gothic ProN",
    "Yu Gothic",
    "Meiryo",
    sans-serif;
}

.traditional {
  font-family:
    "Noto Serif JP",
    "Hiragino Mincho ProN",
    "Yu Mincho",
    serif;
}
```

## 文字サイズ

### 推奨サイズ（A4基準）

| 要素 | サイズ | 備考 |
|------|--------|------|
| キャッチコピー | 20pt以上 | 最も大きく、目立つ |
| 見出し | 14-18pt | 情報ブロックの区切り |
| 本文 | 8-12pt | 可読性確保 |
| 注釈 | 6-7pt | 可読性の下限 |

### 高齢者向け配慮
- 本文: **12pt以上**
- 行間: 広め（2倍）
- 文字色: 濃いめ

### ジャンプ率
文字サイズの比率（ジャンプ率）を大きくとると**メリハリ**が出る。

- **高ジャンプ率**: タイトルと本文の差が大きい → 躍動感、インパクト
- **低ジャンプ率**: サイズ差が小さい → 落ち着き、高級感

## 行間・字間

### 行間（行送り）
- **推奨**: 文字サイズの**1.5〜2倍**
- 狭すぎ → 窮屈、読みにくい
- 広すぎ → 間延び、まとまりがない

### CSS設定例
```css
p {
  font-size: 10pt;
  line-height: 1.7;  /* 文字サイズの1.7倍 */
}
```

### 字間（トラッキング）
- **通常**: デフォルト（0）
- **タイトル**: やや広め（0.05-0.1em）で読みやすく
- **狭すぎ**: 詰まって読みにくい

```css
h1 {
  letter-spacing: 0.08em;
}
```

## 縦書き・横書きの使い分け

### 縦書き
- **印象**: 伝統的、和風、高級感、落ち着き
- **用途**: 和食店、呉服店、寺社仏閣、書道
- **注意**: 数字・アルファベットの扱いに注意

### 横書き
- **印象**: 現代的、カジュアル、情報整理しやすい
- **用途**: IT、スーパー、若者向け、URL含む場合

### CSS縦書き
```css
.vertical {
  writing-mode: vertical-rl;
  text-orientation: mixed;
}
```

## 文字の強調方法

| 手法 | 効果 | 用途 |
|------|------|------|
| 太字 | 目立つ、力強い | キーワード強調 |
| 大きく | 最も目立つ | タイトル、価格 |
| 色変え | 注目を集める | CTA、特典 |
| 下線 | 控えめな強調 | リンク、注目点 |
| 囲み | 独立した情報として認識 | 価格、特典 |
| 影・縁取り | 派手、インパクト | セール、イベント |

## 数字用フォント（Inter）

チラシでは価格・人数・期間などの数字を突出表示する。
数字・英字には **Inter** フォントを使い、日本語テキストと視覚的に差別化する。

```css
.number-emphasis {
  font-family: "Inter", sans-serif;
  font-weight: 900;
  line-height: 1;
}
```

数字の強調サイズについては `references/chirashi-design-system.md` の「数字の強調表現」を参照。

- [ ] 行間は1.5倍以上あるか

- [ ] フォントは3種類以内か
- [ ] 高齢者ターゲットなら12pt以上か
