# チラシデザイン原則

## 1. レイアウトの基本

### グリッドシステム
- **目的**: 要素を規則正しく配置し、デザインに一貫性を持たせる
- **基本**: A4の場合、縦横を3分割または4分割
- **配置**: 重要な要素をグリッドの交点や線上に
- **定番**: 3段組（情報を3ブロックに整理）

### 余白（マージン）
- **版面率**: 紙面全体に対する印刷領域を**60%〜70%**に
- **最低余白**: 仕上がりサイズの四方に**最低10mm以上**
- **グルーピング**: 関連情報は近くに、グループ間に余白を設ける

### 視線誘導パターン

#### Z型（横書きレイアウト基本）
```
①左上 → ②右上
    ↘
③左下 → ④右下
```
- キャッチコピー: 左上
- CTA/結論: 右下

#### F型（情報量多い場合）
```
①━━━━━→
②━━━━→
③━━━→
```
- 各行の左端に重要キーワード
- 拾い読みでも内容が伝わる

#### N型（縦書きレイアウト）
```
①右上
  ↓
②右下 → ③左上
            ↓
          ④左下
```

## 2. 情報の優先度と階層構造

### 5W1Hで情報整理
1. **What** - イベント名/商品名（最重要）
2. **When** - 日時
3. **Why** - 特典/メリット
4. **Where** - 場所
5. **Who** - 対象者
6. **How** - 連絡先/申込方法

### 階層の視覚化
| 手法 | 使い方 |
|------|--------|
| サイズ | 重要→大、それ以外→小 |
| 太さ | キーワードをBold |
| 色 | アクセントカラーで強調 |
| 配置 | 視線誘導の起点に重要情報 |
| 囲み | 特典・価格を枠で強調 |

## 3. キャッチコピーの作り方

### テクニック
1. **ターゲット絞り**: 「〇〇でお悩みの方へ」
2. **ベネフィット提示**: 得られる未来を描く
3. **数字を入れる**: 「満足度98%」「限定50名」
4. **緊急性**: 「今だけ」「本日限り」
5. **簡潔に**: 短く、リズミカルに

### 例
- ❌「当店のラーメンは美味しいです」
- ✅「行列30分！濃厚とんこつ、今だけ半額」

## 4. 写真・イラストの使い方

### 品質基準
- **解像度**: 350dpi推奨（72dpiは不可）
- **明るさ**: 明るくクリアに
- **プロ品質**: 素人写真は避ける

### 役割別の使い方
- **アイキャッチ**: 読み手の注意を引くメインビジュアル
- **共感**: ターゲット層に近い人物（視線の先にCTA）
- **説明**: 商品の使用イメージ、シズル感

### トリミング
- 全体ではなく、最も伝えたい部分を大胆に切り取る

## 5. CTA（行動喚起）の配置

### 位置
- **Z型終点**: 右下
- **または**: 最下部中央

### デザイン
- アクセントカラーの全幅色帯で目立たせる（チラシではボタン風UIは使わない）
- 具体的な行動を示す言葉
- 詳細は `references/chirashi-design-system.md` のCTA・コンタクトバンドを参照

### 必須要素
- **電話番号**: 大きく、ゴシック体で
- **QRコード**: Webサイト/申込フォームへ誘導
- **地図**: 初めての人向けにシンプルに

## 6. 印刷設定

### 解像度
- **商業印刷標準**: 350dpi
- **Web用画像（72dpi）は使用不可**

### カラーモード
- **印刷**: CMYK
- **注意**: RGB→CMYKで色がくすむ（特に鮮やかな青・緑・ネオン）

### 塗り足し（裁ち落とし）
- フチまで色・写真を配置する場合
- **仕上がりサイズ外側に3mm**余分にデザインを広げる
- 断裁ズレで白地が見えるのを防ぐ

### サイズ一覧（塗り足し込み）
| サイズ | 仕上がり | 塗り足し込み |
|--------|----------|--------------|
| A4 | 210×297mm | 216×303mm |
| A5 | 148×210mm | 154×216mm |
| B5 | 182×257mm | 188×263mm |
